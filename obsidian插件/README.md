# DeepSeek 文本优化器

一个基于 DeepSeek API 的 Obsidian 插件，提供文本优化、思维启发和 AI 观点咨询功能。

> 📦 **GitHub 仓库**: 创建仓库后，在此处添加链接

## 功能特性

### 核心功能

1. **文本优化** - 调用 DeepSeek API 优化笔记内容，提升表达流畅度和可读性
2. **思维启发** - 在文本基础上，提供相关的思考启发和延伸建议
3. **AI 观点咨询** - 针对笔记内容提供观点、分析或建议

### 使用方式

- **选中文本处理**：选中要处理的文本，然后执行相应命令
- **整篇笔记处理**：不选中任何文本，将处理整个文档
- **自动替换**：可在设置中开启自动替换原文，或选择生成新版本

## 安装

1. 将插件文件夹复制到 Obsidian 的插件目录：`.obsidian/plugins/deepseek-text-optimizer/`
2. 在 Obsidian 中启用插件
3. 在设置中配置你的 DeepSeek API Key

## 配置

1. 打开 Obsidian 设置
2. 进入「DeepSeek 文本优化器设置」
3. 输入你的 DeepSeek API Key
4. （可选）选择模型和是否自动替换原文

## 可用命令

- **优化文本** - 仅优化文本
- **思维启发** - 仅生成思维启发
- **AI 观点咨询** - 仅提供 AI 观点
- **优化文本 + 思维启发** - 组合功能
- **优化文本 + AI 观点咨询** - 组合功能
- **全部功能** - 同时执行所有功能

## 开发

```bash
# 安装依赖
npm install

# 开发模式（自动监听文件变化）
npm run dev

# 构建生产版本
npm run build
```

## 注意事项

- 需要有效的 DeepSeek API Key
- 使用 API 会产生费用，请注意使用量
- 建议先在小段文本上测试功能
